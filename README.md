# Scrapping Pokédex Pokémon

Ce projet effectue un scrapping du site [pokemondb.net](https://pokemondb.net) pour récupérer les informations et images des Pokémon dans des fichiers CSV.

- Extraction des noms et liens des Pokémon depuis le Pokédex national
- Scrapping des données détaillées et images officielles pour chaque Pokémon
- Résultats enregistrés dans le dossier `csv`

Usage :

- Lancer le script Python `index.py` et suivre les instructions dans le terminal.
